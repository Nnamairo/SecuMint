# SecuMint Documentation

Comprehensive documentation for the SecuMint tokenized securities platform.

## Documentation Structure

### Architecture Documentation
- `architecture/` - System architecture and design decisions
- `api/` - API documentation and specifications
- `contracts/` - Smart contract documentation
- `deployment/` - Deployment guides and procedures

### User Guides
- `user-guides/` - End-user documentation
- `developer-guides/` - Developer integration guides
- `compliance/` - Regulatory compliance documentation

### Technical Specifications
- `specifications/` - Technical specifications and standards
- `security/` - Security protocols and procedures
- `testing/` - Testing strategies and procedures
